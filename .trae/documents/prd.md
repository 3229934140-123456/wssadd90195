## 1. 产品概述

面向医美咨询师团队的企业微信储值风险助手，核心目标是让一线销售在成交前就知道哪些储值方案容易违规，而非事后审计。通过实时风险提示、审批卡点和标准化话术，减少为冲业绩而埋下的退费隐患。

- 目标用户：医美机构咨询师、主管、团队负责人
- 核心价值：将合规风险拦截在成交前，降低退费纠纷和监管处罚概率

## 2. 核心功能

### 2.1 用户角色

| 角色 | 进入方式 | 核心权限 |
|------|----------|----------|
| 咨询师 | 企业微信授权登录 | 客户查询、方案试算、话术留痕、查看个人异常占比 |
| 主管 | 企业微信授权登录 | 以上全部 + 主管审批、查看团队异常占比 |
| 团队负责人 | 企业微信授权登录 | 以上全部 + 团队数据总览、咨询师异常排名 |

### 2.2 功能模块

1. **客户查询**：手机号搜索客户，展示可见储值等级、历史项目偏好、未消耗余额
2. **方案试算**：填写充值金额、赠送比例、绑定项目、承诺有效期，即时风险检测
3. **活动红线**：展示当前活动规则上限、历史违规案例警示、合规指引
4. **主管审批**：特殊折扣/赠金审批流，企业微信消息通知
5. **跟进提醒**：旧卡未消耗提醒、复购周期提醒、待跟进客户列表
6. **话术留痕**：发送储值说明自动带标准条款、风险标签生成、团队异常占比统计

## 3. 核心流程

### 3.1 储值方案创建与风险拦截流程

咨询师从客户查询开始，经过方案试算的风险检测，必要时触发主管审批，最终生成标准化储值说明和风险标签。

```mermaid
flowchart TD
    "A[输入手机号查询客户]" --> "B[查看客户画像与既有套餐]"
    "B" --> "C[创建充值方案]"
    "C" --> "D{实时风险检测}"
    "D" --> "绿色-合规" --> "E[生成储值说明]"
    "D" --> "黄色-注意" --> "F[补充说明后继续]"
    "D" --> "红色-超限" --> "G[提交主管审批]"
    "F" --> "E"
    "G" --> "H{主管审批}"
    "H" --> "通过" --> "E"
    "H" --> "驳回" --> "C"
    "E" --> "I[自动附加标准条款]"
    "I" --> "J[生成风险标签]"
    "J" --> "K[收银台复核]"
```

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深靛蓝(#1B2A4A)代表专业信任感，搭配珊瑚红(#E85D50)作为风险警示色，薄荷绿(#34D399)作为合规安全色
- **辅助色**：琥珀黄(#F59E0B)用于注意级别提示，浅灰蓝(#E2E8F0)用于背景
- **按钮风格**：圆角8px，主要操作按钮带微阴影，危险操作按钮红色高亮
- **字体**：标题使用思源黑体(Noto Sans SC) Bold，正文使用思源黑体 Regular
- **布局风格**：左侧固定导航栏 + 右侧内容区，卡片式模块布局
- **图标风格**：线性图标，2px描边，与整体专业简洁风格统一

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 客户查询 | 搜索栏 | 居中大搜索框，带搜索图标，输入即搜索 |
| 客户查询 | 客户画像卡片 | 左侧头像+等级徽章，右侧信息网格，底部充值时间轴 |
| 方案试算 | 方案编辑器 | 分步表单，步骤条引导，每步带即时校验 |
| 方案试算 | 风险检测面板 | 右侧固定悬浮面板，交通灯式三级指示，展开查看详情 |
| 活动红线 | 规则看板 | 瀑布流卡片，每条规则配图标和数值高亮 |
| 活动红线 | 违规案例墙 | 横向滚动卡片，案例封面+脱敏摘要 |
| 主管审批 | 待审批列表 | 左侧列表+右侧详情的双栏布局，卡片带风险等级色条 |
| 跟进提醒 | 待跟进列表 | 时间轴式排列，紧急程度色标，可滑动操作 |
| 话术留痕 | 储值说明预览 | A4纸样式预览，可折叠标准条款区，一键复制/导出 |
| 话术留痕 | 异常统计图 | 柱状图+趋势折线图组合，咨询师头像排名 |

### 4.3 响应式

- 桌面端优先设计(1440px+最佳体验)
- 平板端(768px-1024px)导航栏折叠为底部Tab
- 移动端(<768px)适配企业微信内置浏览器，单栏布局
